Step by step documentation for install theme and customize theme modules.

Note : Please read carefully and follow the document , before attempting to install the theme, IT IS RECOMMENDED TO ALWAYS BACKUP YOUR CURRENT STORE BEFORE MAKING ANY UPGRADES OR CHANGES !

Installation WayBack to Top

Method 1

Please Log in to your Prestashop Store Admin panel, Go to Design >> Theme & Logo , Now click on "Add new theme" to add theme.

Method 2

Direct Upload modules, themes folder (Provided in our Theme Package) to your prestashop root installation by FTP tool ( ex : FILEZILA, CUTEFTP ) or from your cpanel.

Active Theme

After successfully uploading theme on your current store now you have to active theme from the admin side , Go to Design >> Theme & Logo . From Bottom Select our theme "kenzyfashion" by click on "Use this theme" .

Theme Modules Configuration Back to Top

After activate our theme you need to edit some of theme module for make your store same as our demo store , refer below modules settings.

Go to Modules >> Modules & Services , Installed modules tab > now search and configure all the modules provided by Aeipix.

16. Ps Blog

Find psblog module inside theme package extra module directory , upload module and click on Configure

Find psblog , Click on Configure

After install module you need to edit some height & width , refer below Ps Blog Management settings.

Go to Ps Blog Management >> Blog dashboard >> Listing Blog Setting

Listing Blog Setting

Go to Ps Blog Management >> Blog dashboard >> Item Blog Setting

Item Blog Setting

If you want to change blog description & title settings, refer below settings.

Go to Ps Blog Management >> Blogs Management

To add blog link in main menu

Go to Module >> Main Menu >> Add New Link

add below code after your store url, for example

https://demoprestashop.aeipix.com/AX01/kenzyfashion09/index.php?fc=module&module=psblog&controller=list

?fc=module&module=psblog&controller=list

Make your changes than save .
